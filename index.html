Lovefield

Lovefield is a relational database for web apps.
It is written in JavaScript and works cross-browser.
It provides SQL-like APIs that are fast, safe, and easy to use.

Download

Install via npm

npm install lovefield

Install via bower

bower install lovefield

Or just download zip
